    • Accessibility Statement

    עדכון אחרון:2.1.2026

    Ariel Achon Mortgage and Financial Consulting - Mortgage in Pajamas is committed to making services accessible to the general public. We work to implement the guidelines of the Israeli Standard (SI 5568) based on WCAG 2.1 guidelines at AA level.

    רמת נגישות: AA (תקן ישראלי ת"י 5568)

    1. Accessibility Policy

    We believe in equal rights and in every person's right to receive information and services independently and equally. This website was designed and built with special emphasis on accessibility for people with disabilities.

    2. Website Accessibility Arrangements

    • Keyboard navigation support
    • Screen reader support
    • Appropriate color contrast
    • Flexible and scalable font sizes
    • Detailed image descriptions (Alt text)
    • Hierarchically structured headings
    • Descriptive and clear links
    • Zoom support up to 200% without content loss

    WCAG 2.1 Standard Compliance

    The website complies with WCAG 2.1 AA level standard, which includes:

    • Perceivable: Information is presented in ways users can perceive
    • Operable: Interface components are operable
    • Understandable: Information and user interface are understandable
    • Robust: Content can be interpreted by a variety of technologies

    3. Supported Assistive Technologies

    • NVDA (Windows)
    • JAWS (Windows)
    • VoiceOver (Mac/iOS)
    • TalkBack (Android)
    • Dragon NaturallySpeaking
    • Switch navigation

    4. Accessibility Testing

    The website has been tested using:

    • Automated accessibility testing tools
    • Manual testing with screen readers
    • Keyboard-only navigation
    • Color contrast testing
    • Testing with users with disabilities

    5. Continuous Improvements

    We work continuously to improve website accessibility and update it according to technological developments and user feedback.
